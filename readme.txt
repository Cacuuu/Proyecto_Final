En la pagina inicial se ve el template padre, cargado con datos random, no tiene muchas funcionalidades extra.

Botones:
"Estudios": te lleva al url Estudios.html, en el que figura una tabla con la base de datos del modelo con mismo nombre, y debajo de este, un boton desplegable que permite crear un nuevo estudio.

"Experiencia": te lleva al url Experiencia.html, en el que figura una tabla con la base de datos del modelo con mismo nombre, y debajo de este, un form que permite buscar si un puesto se encuentra en la BD.
